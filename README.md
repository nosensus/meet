# Скрипт продаж · веб-оболочка для отдела продаж

Интерактивный помощник менеджера на основе тренинга Е. Колотилова «Эффективные продажи b2b»
(продукт — b2b-платформа BIZKIM). Сотрудник во время звонка идёт по **каскадным дроп-даунам**:

1. выбирает **тип разговора** (холодный звонок / презентация / возражения / цена / закрытие);
2. при возражении — выбирает **конкретное возражение** (≈10 типов);
3. выбирает **вариант ответа** — появляется готовая реплика «Скажите клиенту»;
4. отмечает **реакцию клиента** — скрипт продолжается дальше.

Все произнесённые реплики собираются в панель **«Сценарий звонка»** (можно скопировать).

## Стек
React 18 · TypeScript · Vite · pnpm.

## Запуск
```bash
pnpm install
pnpm dev      # дев-сервер (http://localhost:5173)
pnpm build    # продакшн-сборка в dist/
pnpm preview  # предпросмотр собранного
```

## Где править контент
Весь скрипт — в одном файле: `src/data/script.ts` (граф узлов `SCRIPT`).
Каждый узел — это шаг с дроп-дауном; у варианта может быть реплика (`say`),
подсказка (`note`) и ссылка на следующий шаг (`next`). Лист с `next: "continue"`
ведёт в универсальный узел продолжения диалога.

Добавить возражение/ответ = добавить узел и опцию в `script.ts`, перекомпиляция не нужна
(дев-сервер подхватит на лету).
